# svg-diagram

Generate pixel-perfect SVG diagrams, banners, and architecture visualizations for READMEs, Notion, and presentations.

## Features
- **Hero Banners**: Project title cards with gradient backgrounds, tech stack pills
- **Architecture Diagrams**: System flow with time labels, pipeline boxes, monitoring sections
- **Flowcharts**: Step-by-step process visualization
- **Pipeline Visuals**: Horizontal/vertical processing pipelines with arrows

## Key Differentiators
- **Anti-overlap rules**: Precise coordinate calculation prevents text/element collisions
- **Dark mode compatible**: Default palette works on both light and dark backgrounds
- **Monospace typography**: Clean, technical aesthetic
- **No external dependencies**: Pure SVG markup, renders everywhere
