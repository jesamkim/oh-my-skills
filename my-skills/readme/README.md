# README Craft - Professional README Generator

Create beautiful, professional README.md files for any project type.
Based on 100+ curated best practices from the [awesome-readme](https://github.com/matiassingers/awesome-readme) community.

## Features

- **8-Step Workflow**: Project analysis, structure selection, header, badges, visual demo, content, TOC, polish
- **7 Project Templates**: Open-source library, CLI tool, web app, API, internal project, ML/data science, monorepo
- **Badge Guide**: Shields.io badges for build status, version, license, coverage, downloads, and more
- **Visual Elements**: GIF demos, Mermaid diagrams, screenshots, dark/light mode compatibility
- **GitHub Features**: Alerts, collapsible sections, task lists, math expressions, footnotes
- **Quality Checklist**: 10-point verification before delivery

## Quick Start

```bash
# Trigger with:
/readme
# or natural language: "README 만들어줘", "create readme", "improve readme"
```

## Structure

```
readme/
  SKILL.md              # Main skill instructions (8-step workflow)
  references/
    structure.md        # 7 project-type README templates
    badges.md           # Shields.io badge reference
    visual-elements.md  # GIF, Mermaid, screenshot guide
  LICENSE               # MIT License
  README.md             # This file
```

## What It Generates

A README tailored to your project type with:

| Element | Description |
|---------|-------------|
| Header | Centered logo/banner + one-line description |
| Badges | Relevant shields based on project type |
| Visual Demo | Screenshot, GIF, or Mermaid diagram |
| Quick Start | Zero-to-running in under 5 commands |
| Usage | Copy-pasteable code examples |
| Architecture | Mermaid diagram for complex projects |
| Contributing | Link to CONTRIBUTING.md |
| License | Clear license statement |
